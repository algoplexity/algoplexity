**What you say:**

> It only shows you what *different observers* think is happening.”

> “If those observers disagree… that’s where the science begins.”

---

**What audience sees:**

* A single system (network animation / swarm / adjacency evolution)

  * Structure LEDs (predicates)
  * Gauges (observers)
  * Resolution slider ( k )
  * Noise slider ( \eta )

---

**What you say:**

> “This is not a simulator.

### Action

### Action

# 🔥 **4. The Main Event: Vicsek System**

### Action

* Switch to **CIO Vicsek generator**
* Slowly increase noise ( \eta )

---

## 🟡 Phase A — Low Noise

### What audience sees

* Agents aligned, moving together
* Gauges:

  * ( L_{lat} ) ↓ (predictable locally)
  * ( L_{sym} ) ↑ (no global repetition)
  * ( \Delta L ) ↑

---

### What you say

> “This looks coordinated to us.”

> “But globally, the system never repeats.”

---

### Key insight

> Local order ≠ Global order

---

## 🔴 Phase B — Increasing Noise

### Action

* Slide ( \eta ) upward slowly

---

### What audience sees

* System destabilizes
* ( r_{eff} ) spikes
* LEDs flicker (predicate transitions)

---

### What you say

> “Now we’re crossing a regime boundary.”

> “The system is no longer stable.”

---

### Key insight

> Transition detection without defining “coordination”

---

## 🟣 Phase C — High Noise

### What audience sees

* Fully disordered
* Observers converge again
* ( \Delta L ) ↓

---

### What you say

> “We’re back to randomness.”
