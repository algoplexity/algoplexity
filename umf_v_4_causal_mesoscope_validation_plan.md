# Unified Mesoscope Framework (UMF v4.0)

## Bounded Algorithmic Causal Mesoscope

---

# Purpose

UMF v4.0 integrates perturbation-based Algorithmic Causal Deconvolution (ACD) directly into the core Mesoscope architecture.

The system now performs:

1. Deterministic transduction
2. Algorithmic causal deconvolution (component separation)
3. Bounded prime-composition generator search per component
4. Description-geometry-based regime detection

The objective is to validate that the system can:

- Separate intertwined generative mechanisms
- Recover minimal prime compositions per component
- Detect structural regime transitions
- Remain bounded and deployable

No statistical metrics are used in the core inference engine.
All inference is description-length and perturbation based.

---

# I. Ontological Model

Observation window W_t may be generated by multiple independent components:

    W_t = ⊕_{i=1}^k G_i

Each component G_i is generated by a bounded prime composition:

    G_i ∈ G_d = { p_m ∘ ... ∘ p_1 | p_j ∈ P, depth ≤ d }

Where:

    P = fixed prime ECA basis (38-rule minimal set)
    d = bounded composition depth

Goal:

Minimize total description length:

    K*(W_t) = Σ_i [ K(g_i) + residual(W_t^i | g_i) ] + K(partition)

Subject to bounded search constraints.

---

# II. Layered Architecture

## Layer 1 — Deterministic Transduction

Analog → Binary tape via fixed lens family.

Approved lenses:
- L1: First derivative sign
- L2: Level bisection
- L3: Second derivative sign

No statistical normalization.

---

## Layer 2 — Algorithmic Causal Deconvolution

For window W_t:

1. Perform perturbation analysis:
   - Remove local element (bit / edge / pixel).
   - Measure ΔK under current model.

2. Terminating criterion:
   Stop partitioning when information change exceeds:

       ΔK > log(2) + ε

3. Partition W_t into independent components:

       W_t → {W_t^1, W_t^2, ..., W_t^k}

Bounded constraints:
- Maximum component count k_max
- Maximum perturbation iterations

Output:
- Component set
- Perturbation sensitivity map

---

## Layer 3 — Bounded Prime Composition Search

For each component W_t^i:

1. Enumerate compositions up to depth d.
2. Compute description cost.
3. Select minimal generator g_i*.

Atomic rule = depth 1 special case.

Return per component:
- g_i*
- depth_i
- margin_i = K₂ - K₁

---

## Layer 4 — Global Description Geometry

Define:

    ρ_t = K*(W_t) / (|W_t| - 2)

Track:
- Generator structure changes
- Component count changes
- Depth escalation
- Margin collapse

Break detection derives strictly from geometry.

---

# III. Break Signal Basis

Primitive break signals:

1. Component Count Shift
       k_t ≠ k_{t-1}

2. Generator Structure Switch
       g_i,t* ≠ g_i,t-1*

3. Depth Escalation
       depth_i,t increases

4. Density Jump
       |ρ_t − ρ_{t-1}| > τ_ρ

5. Margin Collapse
       margin_i < τ_Δ

All signals are description-length derived.

# IV. Validation Phases

---

## Phase 1 — Perturbation Consistency Test

Goal:
Validate that perturbation criterion detects true separable components.

Synthetic Cases:
- Single prime generator
- Single composite generator
- Intertwined Rule 60 ⊕ Rule 110
- Intertwined prime ⊕ composite

Procedure:
- Apply perturbation deconvolution.
- Compare partition to ground truth.

Acceptance:
- Correct component count recovery.
- No spurious partitions on pure systems.

---

## Phase 2 — Composite Generator Recovery

For each recovered component:

- Verify correct prime composition identification.
- Verify minimal depth selection.

Test cases:
- Rule 110 decompositions
- Rule 54 composite form

Acceptance:
- Composite cost < atomic surrogate cost.
- Prime components consistent up to equivalence.

---

## Phase 3 — Structural Break Tests

Scenarios:
1. Component merge
2. Component split
3. Prime → composite transition
4. Depth escalation within component

Measure:
- Detection timing
- Ordering of break magnitudes
- Stability before transition

---

## Phase 4 — Noise Robustness

Inject bounded noise into intertwined systems.

Measure:
- Stability of partition
- False component inflation
- Generator hallucination rate

Acceptance:
- Small noise does not produce spurious components.
- Degradation is gradual, not catastrophic.

---

## Phase 5 — Controlled CPS Deployment

Apply full pipeline:

Transduction → Deconvolution → Prime search → Break geometry

Measure:
- Detection delay
- Interpretability of recovered generators
- Control adaptation benefit

Neural accelerators (if used) must be frozen.
