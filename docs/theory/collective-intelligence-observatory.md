# Collective Intelligence Observatory (CIO) — Canonical Theory Note

## Purpose

This document is the canonical theory note for the Collective Intelligence Observatory (CIO). It records the authoritative statement of the CIO theory, the key terms it introduces, the commitments it makes, and the role of this repository in the RDMP theory chain.

---

## Canonical Statement

> Collective intelligence is the emergence of compressible interaction structure, produced through dynamical processes and measurable in real time.

A multi-agent system exhibits collective intelligence when its joint interaction structure is simpler to describe than the sum of its independently described parts. This reduction in algorithmic description length—compressibility—is the defining signature of coordination.

## Key Terms

| Term | Definition |
|------|------------|
| **Collective intelligence** | Compressible interaction structure emerging from dynamical processes in a multi-agent system |
| **Compressibility** | Reduction in algorithmic description length when agents are described jointly rather than independently |
| **Interaction structure** | The topology of relationships between agents as represented in the interaction graph |
| **Dynamics** | The processes by which interaction structure forms, evolves, and stabilises over time |
| **Structure–dynamics duality** | The principle that collective intelligence has two inseparable components: topology (structure) and the processes that produce that topology (dynamics) |
| **MACYB stack** | The layered architecture of Theory, Computation, and Instrumentation within which CIO is situated |
| **RDMP** | Research Data Management Plan; the governance framework that defines the canonical source hierarchy across repositories |

## Theory Commitments

1. **Compressibility as measure** — Collective intelligence is operationalised as reduction in algorithmic description length across interacting agents. Correlation, consensus, and behavioural alignment are insufficient substitutes.

2. **Process over state** — Collective intelligence is a cybernetic process (`interaction → structure → compressibility → adaptation`), not a static emergent property of individual cognition.

3. **Duality** — Structure (compression in space) and dynamics (compression in time) are separable in measurement but unified in theory. Neither component alone constitutes collective intelligence.

4. **Graph primacy** — Coordination is only recognised when it manifests in the interaction graph. Unstructured synchrony does not qualify.

5. **Falsifiability** — The theory is empirically testable. If coordinated groups do not exhibit increased compressibility in their interaction structure, the theory is falsified.
