# 📜 Paper B — Computation Layer of Observer-Relative Complexity (v3.0)

---

## Abstract

This paper defines the computational layer of the CIO framework as a bounded family of estimators over observer-induced representations.

The purpose of this layer is not to define observable quantities or system ontology, but to approximate latent structural complexity from representations ( \phi_O(X_t) ) generated by a fixed observer.

All computational procedures are epistemically constrained: they operate only on representations and do not access or redefine underlying system states.

---

# 1. Position in CIO Stack

This layer operates strictly between:

```text
Theory (what must hold)
Measurement (what is observed)
```

It defines:

> how hidden structure is approximated, not what exists or what is observed.

---

# 2. Input Space

All computational procedures operate on:

```text
x_t = φ_O(X_t)
```

Where:

* ( X_t ): system state (not accessible)
* ( φ_O ): observer encoding
* ( x_t ): observer representation

No access to ( X_t ) is permitted.

---

# 3. Target Quantity (Latent, Not Observable)

Computation approximates:

```text
K̂_O(x_t)
```

Where:

* ( K̂_O ): observer-relative complexity estimator
* Not directly observable
* Not uniquely defined
* Not a measurement

---

# 4. Computation as an Estimator Family

The computation layer defines a **set of admissible estimators**:

```text
𝒞 = { C₁, C₂, ..., Cₙ }
```

Each estimator:

```text
C_i : φ_O(X_t) → ℝ
```

must satisfy:

* bounded complexity
* deterministic or seeded stochasticity
* observer consistency
* representation invariance under encoding-preserving transforms

---

# 5. Admissible Estimators

The following are valid elements of 𝒞:

## 5.1 Symbolic Compression Estimator

Approximates structure via lossless compression:

* zlib
* LZ77
* entropy coding

---

## 5.2 Algorithmic BDM Estimator

Approximates Kolmogorov-style structure via decomposition:

* structural reuse

---

## 5.3 Neural BDM Estimator

Neural approximation of structural compressibility:

* learned priors over representation space
* bounded capacity model
* trained only on φ_O(X_t)

---

## 5.4 MILS-style Perturbation Estimator

Estimates structural sensitivity via controlled perturbations:

* removal / masking / substitution
* recomputation of estimator output
* difference as structural proxy

---

K̂_O(x_t) ≈ structure under observer O constraints

Important:
